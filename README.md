# Tactical FDC & MGRS (戦術射撃指揮＆読図ナビゲーション PWA)

迫撃砲の射撃指揮（FDC: Fire Direction Center）システムと、MGRS（ミリタリーグリッドリファレンスシステム）を用いた高度な読図・測量ナビゲーションを統合した戦術用PWA（Progressive Web App）です。

電波の届かない演習場や過酷な野外環境での運用を前提としており、一度読み込めば**完全オフライン**で動作します。

## 🌟 主な機能

本アプリは5つの主要モジュールと、戦術的なUIで構成されています。

### 1. 🎯 FDC (射撃指揮)
* **3個分隊の同時計算**: 目標座標と装薬を入力するだけで、3個分隊それぞれの距離、方位角（mil）、射角（mil）を即時計算します。
* **高度な物理エンジン**: 単なる直線距離だけでなく、装薬の最大射程から初速を逆算。さらに設定された「緯度」から**コリオリの力による偏流（横ズレ）**と**弾着時間（ToF）**を自動計算し、方位角の修正値として組み込みます。

### 2. 🔭 測量・観測 (Survey)
* **極座標 (Polar)**: 観測者(FO)からの距離と方位で目標を割り出します。
* **修正射撃 (Adjust Fire)**: 現在の目標に対し、観測者からの左右・遠近のズレ（メートル）を適用します。
* **交会法 (Intersection / Resection)**: 既知の点からの方位角を用いて、未知の目標や現在地を計算します（前方交会法・後方交会法）。
* **トラバース**: 始点からの方位と距離を連続入力し、最終到達点を算出します。

### 3. 📍 座標管理 (Points)
* 端末のGPSを利用して、現在地のMGRS座標（5桁E/N）を即座に取得・登録できます。
* 登録した座標は端末のローカルストレージに保存されます。
* テキストベースでの一括入出力（エクスポート/インポート）に対応しており、部隊間でのデータ共有が容易です。

### 4. 🗺️ 戦術状況図 (Situation Map)
* 陣地、観測者、目標、および登録した座標ピンを2Dグリッド上に自動プロットします。
* **オフラインマップ対応**: 端末内の地図画像を読み込み、左上・右下の座標を指定することで、完全オフラインのカスタム背景地図として利用可能です。
* 1kmおよび100m間隔のMGRS方眼線（グリッド）を自動描画します。

### 5. 🌙 タクティカル・ナイトモード
* 夜間任務中、スマートフォンのバックライトで視力を奪われないよう、ワンタップでUI全体を赤と黒を基調とした「暗視ゴーグル対応モード」へ切り替えます。

## 📂 ファイル構成

本システムは以下の4つのファイルのみで動作する軽量設計です。

* `index.html` : アプリ本体（UI、計算ロジック、スタイルが全て内包されています）
* `manifest.json` : PWAとして認識させるための設定ファイル
* `sw.js` : オフライン動作とバックグラウンド更新を司るService Worker
* `icon3.png` : アプリアイコン（192x192 または 512x512 を推奨）

---

## 🚀 インストール方法 (PWAとしての導入)

このアプリはApp StoreやGoogle Playを経由せず、ブラウザから直接スマートフォンにインストールして使用します。

1. 本ファイル群をWebサーバー（GitHub Pages、Vercel、Netlifyなど）にアップロードします。
2. スマートフォンのブラウザ（iOSならSafari、AndroidならChrome）で該当URLにアクセスします。
3. 以下の手順でホーム画面に追加します。
   * **iOS (Safari)**: 画面下部の「共有ボタン（四角から矢印が飛び出したマーク）」をタップ ＞ **「ホーム画面に追加」**を選択。
   * **Android (Chrome)**: 画面右上のメニュー（︙）をタップ ＞ **「ホーム画面に追加」**を選択。
4. ホーム画面に追加されたアイコン（`icon3.png`）から起動すると、フルスクリーンのネイティブアプリとして動作します。

### 💡 アップデートの仕組み (バックグラウンド更新)
本アプリは起動速度を最優先するため、起動時はローカルのキャッシュデータを読み込みます（圏外でも1秒未満で起動します）。
通信環境がある場合は、裏側で密かに最新バージョンをダウンロードし、**「次回起動時」**に自動でアップデートが適用される仕様（Stale-While-Revalidate）となっています。
